A — Grouped-column table

Excel-like layout. 4 metric groups × 2 sub-columns (Scope / Validated). Dark green header bar, color-coded sub-headers, green TOTAL row. Maximally data-dense — everything at a glance.

B — Toggle Scope/Validated

Simpler 5-column table: Province + 4 metrics. A "Scope / Validated" toggle in the header switches which set of values is shown. Less columns, less cognitive load — but requires an extra click to compare.

C — Province card grid

Each province gets a mini 2×2 card showing all 4 metrics. Scope is the main number; validated shown below with "→". Visually rich but harder to scan across provinces for comparison.

My recommendation: Option A — matches the Excel format you showed, all data visible at once, most useful for quick cross-province comparison. Click to select your preference.
